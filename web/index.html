Run locally

One-shot CLI after setup. Netlify cannot run a stealth browser — use your machine.

cd Project App Workflow
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
playwright install chromium
copy .env.example .env
# Edit .env + personal_data.json
python main.py

Streamlit UI: streamlit run streamlit_app.py

Optional: start Ollama (ollama pull llama3.2) or set an API key in .env.
